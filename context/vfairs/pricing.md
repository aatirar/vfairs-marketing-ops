# vFairs Pricing Reference Document
> **Purpose:** This document is structured for a Lovable-built pricing calculator. It covers all vFairs products, license tiers, included items, overages, and add-ons.

---

## Calculator Architecture Notes (for Lovable)

- **Products are independent.** A sales rep can build a quote by selecting one or more products simultaneously.
- **Virtual Events Platform** is one product with three tiers (Lite, Basic, Standard). The rep picks the tier.
- **All-in-One Platform Packages** is a separate bundled product that sits alongside individual products in the calculator. It should not replace or disable individual product selections.
- **Add-ons are per-product.** If the same add-on (e.g., Integrations at $2,950) appears on multiple selected products, it stacks independently and is charged per product.
- **Overage rates** are displayed as informational text next to the relevant license. No automatic overage calculation is needed — the rep calculates manually.
- **Hardware line items** (Printers/Routers/iPads, Badges & Lanyards) appear as optional checkbox line items, **unchecked by default**. When checked, the rep can enter a custom numeric dollar value. These do not carry a fixed price.

## Global Calculator Rules Summary

| Rule | Behavior |
|---|---|
| Multiple products | Can be selected and quoted simultaneously; totals accumulate |
| Virtual Events Platform tiers | Lite / Basic / Standard are mutually exclusive tiers within one product selection |
| All-in-One Package | Sits alongside individual products; does not replace them |
| Add-ons | Per-product; same add-on selected on two products = charged twice |
| Overage rates | Displayed as informational text next to each license; no auto-calculation |
| Hardware line items | Unchecked by default; check to enable; rep enters a custom dollar amount |
| Multi-year discounts | Note: "Ask about multi-year discount plans" (no fixed rate to calculate) |
| Annual vs. single event pricing | Clearly labeled per license; rep selects which applies |
