Basic Company Info
What is vFairs? 
vFairs is an all-in-one event management platform that powers in-person, hybrid, and virtual events. Our technology empowers companies worldwide to host custom, immersive events and achieve remarkable results. The platform features immersive virtual venues, advanced networking, world-renowned customer service, and many features to help you host unforgettable events.

What kind of events does vFairs use?
vFairs helps organizers host various kinds of events like conferences, trade shows, job fairs, open days, corporate events, and internal events. vFairs supports different event formats like in-person, hybrid, and virtual events.
What products/software we offer?
We offer an all-in-one event management solution for all kinds of events. Our technology includes an immersive virtual event platform, a mobile app, onsite event tech solutions like lead capture and badge printing, event registration and ticketing software. Our SaaS product is built using a cloud-hosted, modular architecture with a multi-tenant database. It provides RESTful APIs that allows developers to extend functionality.

vFairs Badge Printing
The vFairs Badge Printing and Check-in Software offers a seamless, all-in-one solution for in-person event check-ins and badge printing. It is designed to reduce wait times by providing features such as QR code scanning, instant badge printing, and self-check-in options, all through a single app.

●	Customizable Badge Design: Users can create single or double-sided badges with customizable font sizes, colors, and layouts, as well as add QR codes, sponsor logos, and images to make the badges stand out.

We can help source badges based on your badge material and thickness requirement. The solution supports group check-ins and advanced search filters for easy attendee lookup. Attendees can check-n through QR code scans or facial recognition.

●	Self-Service Kiosks: Event organizers can set up branded check-in kiosks with customizable app home screens, offering a quick and easy badge printing experience for attendees.
●	Flexible Printing Options: The platform integrates with various printer types, including Zebra and Epson, to print PVC badges or adhesive labels, and it even supports home printing.
●	Comprehensive Attendee Management: Users can easily manage attendee profiles through the app, sync data between the web and mobile versions, and activate or deactivate profiles with a single click. Walk-in attendees can register and create profiles on the spot.
●	Real-time Data & Analytics: Organizers can track attendance, manage session check-ins, monitor event capacity, and measure badge printing ROI through real-time analytics. Post-event reports are also available in PDF format.
●	CRM Integration: The solution integrates with various CRM and Martech systems, streamlining data management and enhancing post-event follow-up.
Overall, the solution offers a user-friendly interface for both event planners and attendees, ensuring smooth event entry, efficient badge printing, and accurate attendee tracking. For more information visit: https://www.vfairs.com/event-management-platform/event-check-in-and-badge-printing/

Chat Module 
vFairs has an advanced chat module where attendees can talk to each other through 1:1 or group chats. The chat rooms can be moderated and have search filters so attendees can quickly find the person they’re looking for. Attendees can also reply in threads so the conversation can flow easily. Organizers can leave notes on chatrooms for announcements, group chat rules, etc.
Roundtables 
Roundtables are a great way to talk to other people around a similar topic of interest. They allow conversations on one topic so that it remains focused, energetic, and productive. Each roundtable can have a fixed maximum number of attendees who can engage in audio or video chat with each other.
Event Feed 
A live event feed is a great way to boost a sense of community among attendees. They can use the event feed to post about discussion topics, find mutual fields of interests, ask others questions, etc.   
Smart Matchmaking
Attendees can be matched with others based on their interests and preferences through smart matchmaking. Everyone will be asked a fix set of questions and based on their answers they will be matched with their most suitable match for a timed conversation. The module also suggests ice-breaking questions to make it easier for attendees to engage and at the end of each meeting they have the option to increase their time and continue talking to each other.
Spatial Networking
Spatial networking is ideal for virtual events that want to mimic the feel of an onsite event’s networking lounge. Create a dynamic virtual environment that replicates the spontaneity of in-person networking. You can choose what your spatial networking room will look like for example a bookstore, a coffee shop, a park, etc. Attendees can move their avatars around the rooms and talk to or listen to conversations in their close vicinity. If they move away from a group, they won’t be able to take part in the conversation anymore.  
Meeting Scheduler
Attendees and exhibitors can book meetings with each other through a meeting scheduler in exhibit booths. Exhibitors can showcase a few time slots for each rep’s availability and attendees can simply click on one of them to schedule a meeting.
Attendee Profile Search
The process of discovery is simplified with attendee profile search. Audiences can look for people that match their interests through search filters.
