## 1. One-liner

A cross-cutting integration layer that plugs vFairs into existing CRMs (HubSpot, Salesforce, Zoho), video platforms (Zoom, Webex, Microsoft Teams, Restream), payment providers (Stripe, PayPal, Authorize.Net), marketing automation tools (Marketo, Pardot, Mailchimp, ActiveCampaign), and 8,000+ apps through Zapier, plus custom webhooks, RESTful APIs across seven endpoint categories, SSO, and bespoke integration services.

### 3.3 CRM Platforms
- **HubSpot**
- **Salesforce**

### 3.4 Email Marketing and Marketing Automation
- **Mailchimp**
- **ActiveCampaign**
- **Pardot**
- **Marketo**
- **Constant Contact**

### 3.5 Polling and Engagement
- **Slido**

### 3.6 Social Media
- **Walls.io** (social media wall integration)

### 3.7 Translation and Interpretation
- **KUDO**
- **Wordly**

### 3.8 Single Sign-On and Authentication
- **JWT-protocol SSO**
- **SSO with general Identity Provider (IdP)**
- **Google Workspace SSO**
- Use existing credentials from contact databases

### 3.9 File and Report Delivery
- **FTP** (for automated custom report delivery)
- **Amazon S3** (for automated custom report delivery)

### 3.10 Webhooks
- **Custom webhooks** through the vFairs backend
- Webhook API endpoint for automating data syncing
- Notifications to team workspace as actions are taken
- Automate follow-ups after attendees complete certain actions
- Create follow-up tasks in a task management system when an attendee takes action

### 3.11 Zapier (8,000+ Apps)
- **Send attendee registrations** directly to CRM or marketing tools
- **Power sales pipeline** by syncing exhibitor sign-ups automatically
- **Keep databases updated** with speaker registrations in real time
- **Capture and share leads** instantly across favorite business apps
- **Bidirectional sync** patterns supported (e.g. vFairs and Dynamics 365)

### 3.12 RESTful API Suite
Seven API endpoint categories:
1. **Attendee Management** - registration, profile updates, resume uploads
2. **Segmentation Management** - manage segments to target specific groups
3. **Session Management** - organize sessions by categories, dates, themes; supports localization
4. **Exhibitor Management** - manage virtual exhibition booths
5. **Event Management** - retrieve event metadata, available languages, detailed event information
6. **Reports Management** - retrieve event analytics including QR scan analytics and user journey reports
7. **Webhook** - automate data syncing through custom webhooks

### 3.13 Custom Integrations Service
- **Bespoke integration design and build** through partnership with the vFairs team
- Includes custom Zapier integrations, deeper CRM sync, custom data flows, specialized API connections
- Tailored to each customer's unique tech stack

### 3.14 Custom-Reference Integrations (visual references on the page)
The marketing page also displays logos for: 360Learning, Storyline, Docebo, Notion, Moodle, Seeklogo (these appear as custom integration references rather than productized integrations).

## 4. Deployment Options

Integrations are configured per-event in the vFairs backend. SSO, FTP, S3, and custom integrations typically involve the vFairs solutions team during setup. Zapier and API integrations are self-serve once keys are generated.
