# vFairs Marketing Ops

AI-powered marketing toolkit for the vFairs performance marketing team.
Connects Claude Code to live data from Google Ads, Search Console, and GA4.

## Quick start

→ **[Read the full setup guide: SETUP.md](SETUP.md)**

Setup takes ~45–60 minutes the first time. You'll need credential files from Aatir before starting.

## What's included

**Audit skills** (connected to live vFairs data):
- `/google-ads-audit` — Google Ads performance analysis
- `/search-console-audit` — SEO and organic traffic analysis
- `/mql-report` — MQL volume and conversion trends

**28 marketing skills** for copy, CRO, strategy, campaigns, and more.

## After setup

Open a terminal, navigate to this folder, and run `claude`.
Then type any skill command to get started.
